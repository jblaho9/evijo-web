# Pizzeria EVIJO – Web

Website for **MD GASTRO Pizzeria EVIJO**, a family-run Italian restaurant in Sásová, Banská Bystrica, Slovakia. Operating since 2001 across two locations.

## Pages & Sections

- Hero with online ordering CTA
- Opening hours (restaurant & delivery)
- Delivery info & order options
- Menu preview (pizza, pasta, steaks, salads, desserts) — tab-based, JS-rendered
- Daily lunch menu
- Contact & locations with embedded Google Map
- About section

## Stack

Plain HTML, CSS, and vanilla JavaScript. No build step required.
